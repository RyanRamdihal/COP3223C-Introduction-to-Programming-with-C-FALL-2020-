# Program Assignment 3 - Arrays, Pointers, and File Processing

## Overview
This program reads input data from a file, calculates the slope and y-intercept of a line using two ordered pairs, computes multiple sets of y-values from different x-coordinate arrays, and writes the formatted results to an output file.

The program demonstrates the use of arrays, pointers, loops, functions, and file processing in C.

---

## Assignment Description
This assignment focused on:
- Reading from input files
- Writing formatted output files
- Using arrays and pointers
- Passing arrays into functions
- Reusing functions for calculations
- Computing y-values from multiple sets of x-coordinates

The program:
- Reads ordered pairs from a file
- Computes the slope and y-intercept
- Generates multiple sets of ordered pairs
- Stores computed y-values in arrays
- Outputs formatted coordinate data into a file
