# Quadratic Equation Solver

## Overview
This program solves quadratic equations in C using the quadratic formula. It calculates:
- Real roots
- Repeated roots
- Complex roots

depending on the value of the discriminant.

## Technologies Used
- C
- math.h library

## Features
- Calculates discriminant
- Handles positive, negative, and zero discriminants
- Uses functions for calculations
- Prints roots of the equation

## Example Equation
ax^2 + bx + c = 0

## How to Compile
```bash
gcc quadratic_solver.c -o quadratic_solver -lm
```

## How to Run
```bash
./quadratic_solver
```
