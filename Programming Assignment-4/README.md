# Program Assignment 4 - Structures and Binary Files

## Overview
This program reads slope-intercept data and x-coordinate values from an input file, stores the information inside a data structure, calculates y-values using the slope-intercept equation, and writes the completed data into a binary output file.

The assignment focused on:
- Structures
- Arrays inside structures
- Binary file writing
- Functions
- File processing

---

## Assignment Description
This assignment required:
- Reading data from an input text file
- Storing values in a data structure
- Using arrays within structures
- Computing y-values from x-values
- Writing structured data into a binary file

The program stores:
- Titles
- Slope values
- Y-intercepts
- Arrays of x-values
- Arrays of computed y-values

for multiple data sets.
