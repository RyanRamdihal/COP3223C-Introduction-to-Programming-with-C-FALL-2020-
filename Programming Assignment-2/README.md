# Program Assignment 2 - Quadratic and Linear Equation Solver

## Assignment Description
This assignment builds upon Programming Assignment 1 by expanding the quadratic equation solver program.

Requirements for this assignment included:
- Following proper coding conventions
- Prompting the user for values of `a`, `b`, and `c`
- Correctly solving equations even when `a = 0`
- Allowing the user to repeatedly enter new sets of values using loops

The program solves quadratic equations using the quadratic formula and also handles linear equations when the coefficient `a` equals 0.

## Features
- User input for coefficients
- Real root calculation
- Complex root calculation
- Linear equation solving
- Repeating calculations through loops
- Yes/No input validation
